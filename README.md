<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple example steps.
    <br>
    <br>
### Prerequisites
You will need:
1. C preprocessor
2. C compiler 

* To  install cpp
  ```sh
  apt-get install cpp
  ```
 

* To  install gcc
  ```sh
  apt-get install gcc
  ```
  
    <br>
    <br>
    
### Installation



1. Clone the repo
   ```sh
   git clone https://github.com/Jozmon/ip-dec2bin-private
   ```
2. Compile 
   ```sh
   gcc ip-dec2bin.cpp -o ip-dec2bin
   ```

<!-- USAGE EXAMPLES -->
## Usage

After the compilation to run it use command with bash on linux or cmd on windows.
  ```sh
./ip-bin2dec
   ```
